STOVE Gaming Store v1.0.0 - Windows Installer (테스트 빌드)
============================================================

이것은 테스트용 설치 파일입니다.

시스템 요구사항:
- Windows 10 이상
- 8GB 이상 RAM
- 500MB 디스크 공간

설치 방법:
1. 파일을 다운로드합니다
2. 설치 파일을 실행합니다 (관리자 권한 필요)
3. 설치 마법사를 따라 완료합니다
4. 바탕화면의 STOVE 아이콘을 클릭하여 실행합니다

문제 해결:
- 설치 중 오류가 발생하면 재부팅 후 다시 시도하세요
- 방화벽 설정을 확인하세요
- 프로젝트 이슈 탭에서 문제를 보고하세요

감사합니다!
